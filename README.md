**你是 MCT 吗？** - 查看 [MCT GitHub 用户指南](https://microsoftlearning.github.io/MCT-User-Guide/)

任何 MCT（微软认证培训师）都可以对 GitHub 转载的代码或内容提交拉取请求。 Microsoft 和课程作者将根据需要进行分类，包括内容和实验室代码更改。
